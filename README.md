The frontend of the **Library Management System** is built with **React.js**, providing a modern, responsive UI for managing books and library members, and communicating with the backend API.

## Features

* User-friendly interface for **adding, editing, deleting** books and members
* **Search** functionality for both members and books
* Integration with the backend for seamless data sync
* Secure and intuitive form handling

## Technologies Used

* React.js
* React Router
* Axios
* HTML & CSS

## Setup & Run Locally

1. **Clone the repo:**

   ```bash
   git clone https://github.com/adapakapapinaidu/Lms_Frontend.git  
   cd Lms_Frontend  
   ```

2. **Install dependencies:**

   ```bash
   npm install  
   ```

3. **Configure API endpoint:**

   * Open `src/api/api.js`
   * Change the base URL to your backend API.
     Example:

     ```js
     const API_URL = "https://library-management-system-fixa.onrender.com/";
     ```

4. **Run the frontend:**

   ```bash
   npm start  
   ```

5. **Open in browser:**
   Visit [http://localhost:3000](http://localhost:3000)
